## Figma

- **AI Agents Can Now Write Directly to Figma Files** — Via the Figma MCP server, AI agents can create and modify real design assets using your components, variables, and tokens. - **AI Image Tools Land in FigJam, Slides & Buzz** — Expand, Erase, Isolate, and Vectorize are now available across FigJam, Slides, and Buzz (beta), so you can refine visuals from brainstorm to presentation to promotion without breaking flow. [Source](https://fig-events.figma.com/feb-rn-2026/)

- **Make AI Administration + Eyedropper Upgrade** — Admins can now purchase AI credits with flexible pay-as-you-go or subscription options for Org and Enterprise plans. Plus, the macOS desktop eyedropper can now sample colors from anywhere on screen — not just the canvas. [Source](https://releasebot.io/updates/figma)

## Claude / Anthropic

- **300K Output Tokens Now Available on Batches API** — Anthropic raised the max_tokens cap to 300K on the Message Batches API for Claude Opus 4.6 and Sonnet 4.6, unlocking longer single-turn outputs for long-form content, structured data, and large code generation tasks. [Source](https://docs.anthropic.com/en/release-notes/overview)

- **Models API Returns Capability Fields** — GET /v1/models now surfaces max_input_tokens, max_tokens, and a capabilities object per model — useful for teams building AI-assisted design tooling or product workflows that need to know model limits at runtime. [Source](https://docs.anthropic.com/en/release-notes/overview)

- **Claude Haiku 3 Retiring April 19** — Haiku 3 (claude-3-haiku-20240307) is being deprecated April 19; migrate to Haiku 4.5. The 1M token beta for Sonnet 4.5 and Sonnet 4 also ends April 30 — Sonnet 4.6 and Opus 4.6 now support full 1M context at standard pricing. [Source](https://siliconangle.com/2026/03/27/anthropic-launch-new-claude-mythos-model-advanced-reasoning-features/)

## Design Trends

- **Vibe Design Goes Mainstream: Texture, Motion & "Main Character" UI** — 2026 interfaces are leaning hard into tactile digital aesthetics — jelly/clay morphism, kinetic typography, and Y2K-nostalgic patterns. The goal is emotional resonance over pure function. This is the Vibe Design moment you've been tracking. [Source](https://medium.com/@punitweb/ux-ui-design-trends-for-2026-from-ai-to-xr-to-vibe-creation-7c5f8e35dc1d)

- **AI as Copilot, Not Autopilot — NNGroup's State of UX 2026** — Nielsen Norman Group's annual report highlights the defining UX shift of 2026: AI should feel *present, optional, and respectful of human context* rather than all-knowing. Designing trust-aware AI interactions is the new senior-level UX skill. [Source](https://www.nngroup.com/articles/state-of-ux-2026/)

- **Generative UI + AI Design Systems Emerging as Core Practice** — Designing for intent, Machine Experience (MX) design, and AI-generated design systems are emerging as foundational competencies. Interfaces will increasingly be generated per-user, per-context — which fundamentally changes how we think about component libraries and token architecture. [Source](https://www.stan.vision/journal/ux-ui-trends-shaping-digital-products)
